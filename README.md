# Daksha 2.0

The official event website for IEEE SB CEC.

## About

Daksha 2.0 is a centralized platform for showcasing event information, schedules, and announcements. It provides participants with seamless access to event registration.

## Features

- Event information and schedules
- Live announcements
- Event registration form
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Explore the event details and register
